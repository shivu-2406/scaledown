## Overview
This project is an Educational Content Assistant built using Streamlit and the ScaleDown library.
It helps students by compressing and simplifying long educational content while preserving the correct answer.

## Features
- Accepts long text input
- Removes unnecessary text
- Preserves key information
- Simple Streamlit-based interface

## Tech Stack
- Python
- Streamlit
- ScaleDown
- GitHub
